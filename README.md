

# 🧑‍💻 AI Attendance System

This project is an **AI-powered attendance system** that uses **facial recognition** to automate and streamline student attendance tracking.

### 📌 **Key Features**

* Uses deep learning and computer vision to detect and recognize faces.
* Automates attendance marking from a single classroom photo.
* Prevents proxy attendance and reduces manual effort.
* Generates editable attendance reports for verification.

### ⚙️ **Tech Stack**

* **Deep Learning**: Pre-trained face recognition models (e.g., ArcFace)
* **Computer Vision**: Face alignment and preprocessing
* **Backend**: Python & Flask
* **Database**: PostgreSQL 

### 🚀 **How It Works**

1. Register student faces and store embeddings securely.
2. Teacher uploads a class photo after each lecture.
3. System detects and verifies faces, marking attendance automatically.
4. Generates reports that can be reviewed and updated if needed.
